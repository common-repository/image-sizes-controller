=== Description ===

Get more control over the image sizes being created on your WordPress website. You can create unlimited image sizes, choose the width, height and crop. you can disable any image size from being created for new uploaded images.
The plugin offers the option to disable "BIG image" threshold done by WordPress for images above 2560px in width or height.
The plugin wil list all the image sizes inside the image edit page along with the size image dimensions and its file size.
The list will display if any of the sizes is broken which means the size image is missing and you can fix it in a single click.

== Features ==
* Create custom subsizes.
* Create any missing image subsize.
* Disable any image size.
* Delete image subsize.
* List all image subsizes in image edit page.
* Disable the wordpress [ "BIG image" threshold ]
